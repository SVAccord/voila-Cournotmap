Project: IRM

Function: Visualization

Description: Web application for visualizing Cournot maps*. A tool for determining the number of equivalence groups within the IRM project

*A Cournot map (Carnot cube, Carnot diagram, or Weitch map) is a graphical representation of Boolean functions for the purpose of convenient and visual manual minimization. It is one of the equivalent methods for describing or defining logical functions, along with a truth table or Boolean algebra expressions. A simple algorithm converts a Karnaugh map into a truth table or Boolean formula and vice versa.
